# 📊 Data Information

## 📌 Source
The dataset used in this project is from:

**NSSO 80th Round Modular Telecom Survey (2025)**

---

## ⚠️ Data Availability
Due to data usage restrictions, the raw dataset is not publicly uploaded in this repository.

---

## 🔗 How to Access the Data
You can obtain the dataset from the official source:

👉 https://mospi.gov.in/

---

## 📁 Required Files
To reproduce this analysis, download:

- **CMST80HH** → Household dataset  
- **CMST80PER** → Person-level dataset  

---

## 🔁 Reproducibility Note
This project is fully reproducible given access to the NSSO unit-level data.

All data cleaning, transformation, and modeling steps are provided in the `code/` folder.
